# HTML Profile Card

A semantic HTML profile card project showcasing essential web development skills.

## Features & HTML Skills Demonstrated

### HTML Structure

- Proper document structure with `<!DOCTYPE>`, `<html>`, `<head>`, and `<body>` tags
- Meta tags for character encoding and viewport responsive design
- Semantic HTML5 elements for better accessibility and SEO

### Semantic Sections

- `<header>` - Profile image section
- `<main>` - Primary content container
- `<article>` - Profile card wrapper
- `<section>` - Skills and profile information areas
- `<nav>` - Social links navigation
- `<footer>` - Copyright information

### Headings

- `<h1>` - Primary heading (Profile name)
- `<h2>` - Secondary headings (Role, section titles)

### Paragraphs

- `<p>` - Profile introduction and footer text
- Proper text content with meaningful information

### Lists

- **Unordered lists `<ul>`** for:
  - Skills display
  - Social media links
- **List items `<li>`** for each skill and social link

### Links

- External links with `target="_blank"` and `rel="noopener noreferrer"` for security
- Email link using `mailto:` protocol
- Proper link accessibility with descriptive text

### Images

- Profile image with:
  - Descriptive `alt` attribute for accessibility
  - Circular design with CSS border-radius
  - Hover animation effects

## Features

- **Responsive Design** - Mobile-first approach with media queries
- **Interactive Elements** - Hover effects on cards, images, and buttons
- **Accessibility** - Alt text, semantic HTML, focus states
- **Modern Styling** - Gradient backgrounds, smooth transitions
- **Professional Layout** - Clean, centered card design
